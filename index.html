Installing DappHub’s developer tools for Ethereum

Use the following commands to install the suite of Ethereum developer tools currently recommended by DappHub: 

# Install the Nix package manager:
curl https://nixos.org/nix/install | sh
# Add the DappHub channel to Nix (optional):
nix-channel --add https://nix.dapphub.com/pkgs/dapphub
nix-channel --update
# Install the various developer tools:
nix-env -i dapp hsevm seth solc

See the user manual or join DappHub.chat for more information.
